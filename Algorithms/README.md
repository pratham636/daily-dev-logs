# 🧮 Algorithms Collection (DAA)

This folder contains my implementations of classic algorithms from my **Design and Analysis of Algorithms (DAA)** coursework. These exercises focus on efficiency, time complexity, and problem-solving logic.

## 📂 Included Algorithms

| Algorithm | Language | Description |
| :--- | :--- | :--- |
| **Convex Hull** | C++ | Finds the smallest convex polygon that encloses all given points. |
| **Huffman Coding** | C | A lossless data compression algorithm using a greedy approach. |
| **Matrix Multiplication** | C++ | Standard implementation of multiplying two matrices. |
| **Largest Sub-array** | C++ | Solves the maximum subarray sum problem. |

## 🚀 How to Run
To compile and run any of these files (e.g., Convex Hull), use a C++ compiler like `g++`:

```bash
g++ convex_hull.cpp -o convex_hull
./convex_hull
